# Predicci-n-Precios-Acciones-NVDA-ARIMA
Este proyecto realiza un análisis de series temporales y pronóstico de precios de cierre de las acciones de NVIDIA utilizando un modelo ARIMA. Se descarga datos históricos de 7 años, se realiza un análisis exploratorio, se ajusta el modelo y se generan predicciones para los próximos 12 meses, incluyendo visualizaciones y métricas de evaluación
